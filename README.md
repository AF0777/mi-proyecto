# Proyecto de Geometría Básica

## Descripción
Un programa sencillo en Java para calcular áreas de diferentes figuras geométricas planas empleando funciones.

## Lenguaje
Java

## Funciones
- `calcularAreaRectangulo`: Recibe la base y la altura para retornar el área de un rectángulo.
- `calcularAreaTriangulo`: Recibe la base y la altura para calcular y retornar el área de un triángulo.

## Como ejecutar
Para compilar y correr el programa usa:
javac Geometria.java && java Geometria
